# fpga_support_plug

在vscode上的fpga开发插件
包含了对fpga开发过程中所需要语言的支持
有如下：vhdl，Verilog，systemvVerilog，TCL，xdc，sdc
截止于2020/2/16已完成如下
1：对vivado的开发支持
2：语法自动校测
